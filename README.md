# 📝 Todo App
 
A simple and intuitive Todo application built using **React** for the UI, **Redux** for state management, and custom **CSS** for styling. This app allows users to **add**, **edit**, and **delete** their todos efficiently.
 
---
 
## 🚀 Features
 
- ✅ Add new todos
- ✏️ Edit existing todos
- 🗑️ Delete todos
- 💾 State managed using Redux
 
---
 
## 🛠️ Tech Stack
 
- **React** – Frontend library for building user interfaces
- **Redux** – State management library
- **React-Redux** – Official React bindings for Redux
- **Styling** – AntD
